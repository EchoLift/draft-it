Documentation

Write from cards to screenplay.

Draft It is a card-first screenwriting tool. Start with projects, define characters, plan scenes as movable cards, turn those cards into screenplay pages, and export when the draft is ready.

Quick Start

  1. Create a project from the Projects home screen with title and author name.
  2. Open Characters and create the people who will drive the story.
  3. Use the Character Map to map relationships between character nodes.
  4. Open Cards and add scene cards with title, scene beat, and emotion.
  5. Arrange cards freely on the board or reorder them from the Order drawer.
  6. Open Script, select a card, and write the actual screenplay for that scene.
  7. Turn on auto backup or export a project .draftit backup file often.
  8. Select the cards you want, choose export options, and print the screenplay PDF.
Important Backup Warning

Your browser storage is not a permanent archive.

Draft It saves projects locally in your browser. If browser cache, site data, private browsing data, or device storage is cleared, your drafts can be lost from this browser.

Use Export on each project card at short intervals, especially after important writing sessions. Better: turn on Auto backup active project every 5 min from the Projects page so Draft It downloads regular project backup files while you work.

The Layers

01

Projects

Projects are the home layer. Create unlimited projects for separate films, episodes, scenes, or drafts. Each project keeps its own cards, characters, relationship map, author name, and screenplay text.

02

Characters

Characters store gender, desire, fear, lie, shade, and screen time for each person in a project. Use them as the emotional engine before building scenes.

03

Character Map

The map is a relationship canvas. Add character nodes, connect them with directional relationship lines, zoom out, and remove mistaken nodes or lines without deleting characters.

04

Cards

Cards are the planning layer. Add a title, write the scene beat, choose the intended emotion, and place the card anywhere on the whiteboard.

05

Screenplay

Screenplay is the writing layer. Pick a card from the list, seed the editor with the scene idea, format lines, and write the finished scene.

Writer’s Block Rescue

Use Draft It to make the next scene inevitable.

Writer’s block often happens when the story is treated as “what should happen next?” Draft It helps you switch the question to “what would this character do when trapped in this situation?” That is easier to answer because it comes from desire, fear, and belief.

1. Define the character

Start in Characters. Write what they want, what they fear, and the lie they believe. This gives every scene a pressure point.

2. Make cards as situations

In Cards, do not worry about perfect plot. Create situations: a call, a reveal, a refusal, a public embarrassment, a secret, a choice.

3. Throw them in

Pick a card and imagine the character inside it. Ask how they react, respond, hide, attack, confess, avoid, or behave.

4. Write only that card

Open Script and write the screenplay for that one card. Because every card has its own screenplay, you can move forward without solving the whole film at once.

Working With Cards

Add a card

On desktop, use the left editor. On mobile, tap the + button at the bottom right. Add a title, a scene beat, and an emotion color.

Move and zoom

Drag cards around the board. Drag empty board space to pan. Use the zoom toolbar to zoom in, zoom out deeply, or reset the board view. The Cards header also gives quick access to Projects, Characters, and Script.

Read full scenes

Cards show a short preview so the board stays readable. Tap or click a card to expand the full scene. Close it with × or by tapping outside.

Edit or delete

On desktop, use the card buttons. On mobile, long-press a card to open edit and delete. Delete actions ask for confirmation because they cannot be undone.

Building Characters

Create a character

Use the Characters button from a project, Cards, or Script. Tap +, then add the name, gender, biggest desire or goal, biggest fear, and the lie the character believes.

Choose shade and screen time

Shade can be White, Gray, or Black. Screen time can be Long, Medium, or Cameo. These labels help you quickly read how a character functions in the story.

Use the character picker

Characters appear as horizontal thumbnails. Pick one to view the brief and keep their inner conflict visible while planning scenes.

Export character briefs

During export, choose whether character briefs should print after the title page. Select all characters or only the ones needed for that draft.

Character Map

The Character Map helps you see who is connected to whom before you write a scene. It works like a relationship board for character nodes.

Open the map

Open Characters, then tap MAP. The map opens as a full canvas with its own Projects, Cards, Script, and Characters navigation.

Add character nodes

Choose a character and tap Add node. A character can appear more than once as separate nodes, useful when you want to explore different relationship clusters.

Connect from-to

Pick a From node, a To node, type the relationship, then connect. Arrows show direction, so “mother to son” and “son to mother” are visually clear.

Edit or remove lines

Click the relationship text to rename it. Click the small minus beside a relationship label to remove only that line. Character nodes remain on the map.

Remove nodes

Use the minus on a node to remove only that node box from the map. The character stays saved in the project. Connected lines for that node are removed after confirmation.

Zoom and pan

Use the map zoom controls, mouse wheel, trackpad, or Ctrl/Cmd zoom shortcuts. The map can zoom far out like the Cards board, and dragging nodes works while zoomed.

Scene Emotions

Emotion colors help writers and directors understand how a scene should feel during narration, blocking, and shoot planning.

  • Warmth
  • Tension
  • Relief
  • Hope
  • Romance
  • Mystery

Writing The Screenplay

Select a card

In the Script layer, choose a card from Available cards. Each card stores its own screenplay text, so scenes stay connected to the planning board.

Use scene

Tap Use scene to copy the card’s scene beat into the screenplay editor as a starting point.

Format lines

Use the format toolbar for Scene, Action, Character, Dialogue, Parenthetical, Transition, and Shot lines. Export uses screenplay-style widths for dialogue and right-aligned transitions.

Export

Select the cards you want in the export, then tap Export. Choose whether to include character briefs. Draft It creates a title page first, optional character pages next, and then the selected screenplay scenes in order.

Mobile Guide

Mobile keeps the active canvas as the main screen. The Cards, Map, and Script pages keep their navigation in the top header. Use the bottom-right + button on Cards to add cards, and the cards button to manage order. Long-press a card for edit and delete.

The app remembers the last main page. If you refresh while writing in Cards, Script, or Character Map, Draft It returns to that same page as long as the active project still exists.

Data And Storage

Draft It stores work locally in your browser. Use the same browser and device to continue a draft. Export important work regularly as a .draftit file if you need a backup outside the browser.

Manual backup

On the Projects page, tap Export on the project card you want to back up. Keep that file somewhere safe. It contains that project’s cards, characters, relationship map, author, and screenplay text.

Auto backup

Turn on Auto backup active project every 5 min to download a backup for the active project while the app is open. This is the safest habit during long writing sessions.

Restore

Use Import project to restore a saved project backup. Import adds the project to this browser, or updates the matching project if the backup came from the same project id. Other projects stay untouched.
