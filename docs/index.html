Documentation

Write from cards to screenplay.

Draft It is a card-first screenwriting tool. Start with projects, plan scenes as movable cards, turn those cards into screenplay pages, and export when the draft is ready.

Quick Start

  1. Create a project from the Projects home screen.
  2. Open Cards and add scene cards with title, scene beat, and emotion.
  3. Arrange cards freely on the board or reorder them from the Order drawer.
  4. Open Script, select a card, and write the actual screenplay for that scene.
  5. Select the cards you want and export the screenplay.

The Three Layers

01

Projects

Projects are the home layer. Create unlimited projects for separate films, episodes, scenes, or drafts. Each project keeps its own cards and screenplay text.

02

Cards

Cards are the planning layer. Add a title, write the scene beat, choose the intended emotion, and place the card anywhere on the whiteboard.

03

Screenplay

Screenplay is the writing layer. Pick a card from the list, seed the editor with the scene idea, format lines, and write the finished scene.

Working With Cards

Add a card

On desktop, use the left editor. On mobile, tap the + button at the bottom right. Add a title, a scene beat, and an emotion color.

Move and zoom

Drag cards around the board. Drag empty board space to pan. Use the zoom toolbar to zoom in, zoom out, or reset the board view.

Read full scenes

Cards show a short preview so the board stays readable. Tap or click a card to expand the full scene. Close it with × or by tapping outside.

Edit or delete

On desktop, use the card buttons. On mobile, long-press a card to open edit and delete. Delete actions ask for confirmation because they cannot be undone.

Scene Emotions

Emotion colors help writers and directors understand how a scene should feel during narration, blocking, and shoot planning.

  • Warmth
  • Tension
  • Relief
  • Hope
  • Romance
  • Mystery

Writing The Screenplay

Select a card

In the Script layer, choose a card from Available cards. Each card stores its own screenplay text, so scenes stay connected to the planning board.

Use scene

Tap Use scene to copy the card’s scene beat into the screenplay editor as a starting point.

Format lines

Use the format toolbar for Scene, Action, Character, Dialogue, Parenthetical, Transition, and Shot lines. Export uses screenplay-style widths for dialogue and right-aligned transitions.

Export

Select the cards you want in the export, then tap Export. Draft It opens a print-ready screenplay page that can be saved as PDF from the browser print dialog.

Mobile Guide

Mobile keeps the board as the main screen. Use the bottom-right + button to add cards and the cards button to manage order, move to Projects, or open Script. Long-press a card for edit and delete.

Data And Storage

Draft It stores work locally in your browser. Use the same browser and device to continue a draft. Export important work regularly if you need a backup outside the browser.
